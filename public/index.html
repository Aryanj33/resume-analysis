API Documentation

1. Login API

Endpoint: POST /api/auth/login

Example:

curl -X POST https://resume-analysis-api.onrender.com/api/auth/login \
-H "Content-Type: application/json" \
-d '{
    "username": "naval.ravikant",
    "password": "05111974"
}'

2. Resume Analysis API

Endpoint: POST /api/resume/enrich

Requires Authentication Token from Login API

Example:

curl -X POST https://resume-analysis-api.onrender.com/api/resume/enrich \
-H "Authorization: <token>" \
-H "Content-Type: application/json" \
-d '{
    "raw_text": "Your resume text here"
}'

3. Name Search API

Endpoint: POST /api/search/name

Example:

curl -X POST https://resume-analysis-api.onrender.com/api/search/name \
-H "Content-Type: application/json" \
-d '{
  "name": "name here"
}'
